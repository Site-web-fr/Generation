## Activer GitHub Pages (une seule fois)

Si les liens renvoient une 404, dans le repo GitHub :

1. **Settings → Pages**
2. Source : **Deploy from a branch**
3. Branch : **gh-pages** / **/ (root)**
4. Save

Le workflow pousse automatiquement sur `gh-pages` à chaque push sur `main` ou `cursor/premium-landings-8794`.
