⚠️ DÉMONSTRATION AUDIT SÉCURITÉ — Autorisation client — Ne pas utiliser à des fins malveillantes

Preuve : le login adhérents peut être piégé (clickjacking)

Le site aabv.fr ne envoie pas X-Frame-Options ni CSP frame-ancestors. Un attaquant peut superposer une fausse interface par-dessus le vrai formulaire de connexion.

🎁 Félicitations ! Cliquez pour recevoir votre lot

Ce que voit le client : l'iframe charge le vrai site (formulaire « Espace adhérents » visible en filigrane). Sans protection, un attaquant positionne le bouton « Connexion » exactement sous le bouton leurre.

Correction : ajouter Header set X-Frame-Options "SAMEORIGIN" dans Apache.
