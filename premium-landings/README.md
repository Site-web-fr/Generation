# Premium Landings — Showcase Client

Collection de **10 landing pages premium** (~10 000 € chacune) — démonstrations interactives pour prospection client haut de gamme.

## Stack

- **React 19** + TypeScript + Vite
- **Three.js** (@react-three/fiber) — backgrounds 3D, showroom véhicules
- **Framer Motion** — animations & micro-interactions
- **Configurateurs sur-mesure** — devis, réservation, estimation en autonomie

## Lancer en local

```bash
cd premium-landings && npm install && npm run dev
```

## Build production

```bash

```
