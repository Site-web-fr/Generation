# Premium Landings — 10 Sites d'Exception

Propositions commerciales ultra-premium pour client exigeant. Budget 10 000€/site.

## Stack Technique

- **React 19** + **TypeScript** + **Vite**
- **Three.js** + animations 3D WebGL natives
- **Framer Motion** pour les transitions UI
- **Tailwind CSS** pour le système de design
- **Google Fonts** (Cormorant Garamond, DM Sans, Space Grotesk)

## Lancer le projet

```bash
cd premium-landings
npm install
npm run dev
```

Ouvrir [http://localhost:5173](http://localhost:5173)

## Build production

```bash
npm run build
npm run preview
```

## Design System

- **Couleur or** : `#c9a96e` (Gold)
- **Fond obsidienne** : `#0a0a0a`
- **Typographie principale** : Cormorant Garamond (serif élégant)
- **Typographie corps** : DM Sans
- **Animations** : Framer Motion + Three.js WebGL
- **Curseur custom** : Point doré + cercle suiveur

## UX Principles

Chaque site applique les mêmes principes premium :
1. **Hero immersif** — 3D + parallax + particules
2. **Navigation fixe** — Semi-transparente, backdrop blur
3. **Outil interactif central** — L'utilisateur fait son estimation sans appeler
4. **Social proof** — Stats, témoignages, certifications
5. **CTA multiples** — Floating, inline, footer
6. **Performance** — Canvas WebGL optimisé, images lazy-loaded
